前端自动化Gulp工具常用插件

前端工程化
  • gulp
  • 前端模块化工具

前面文章中提过了Gulp的入门使用，那么这一篇会写一些常见插件的使用。

npm init命令初始化当前文件夹后，在当前文件夹新建gulpfile.js文件。当前目录下的所有操作流都在gulpfile.js文件中定义。

一、gulp-uglify （JS压缩）

gulp-uglify安装:

npm install --save-dev gulp-uglify

gulp-uglify用来压缩js文件，使用的是uglify引擎。

小技巧，第二个参数’！js/*.min.js’是用来过滤掉后缀为min.js，！感叹号为排除模式。

二、gulp-minify-css（CSS压缩）

gulp-minify-css安装：

npm install --save-dev gulp-minify-css

可以使用它来压缩CSS文件

三、gulp-minify-html（html压缩）

gulp-minify-html安装：

npm install --save-dev gulp-minify-html

可以使用它来压缩html文件.

四、gulp-jshint（JS代码检查）

gulp-jshint安装：

npm install --save-dev gulp-jshint

可以用来检查JS的代码

五、gulp-concat（文件合并）

gulp-concat安装：

npm install --save-dev gulp-concat

合并CSS与JS文件，减少http请求。

六、gulp-less（编译Less）

gulp-less安装：

npm install --save-dev gulp-less

把less文件转换为css。

七、gulp-sass（编译Sass）

gulp-sass安装：

npm install --save-dev gulp-sass

把scss文件转换为sass文件。

八、gulp-imagemin（压缩图片）

gulp-imagemin安装：

npm install --save-dev gulp-imagemin

可以使用gulp-imagemin的插件来压缩jpg、png、gif等图片。（imagemin也是有插件的，是基于imagemin产生的插件，所以前缀是imagenin开头）

压缩png插件-imagemin-pngquant安装：

npm install--save-devimagemin-pngquant

gulpfile.js:

gulp-imagemin的使用比较复杂一点，它本身也有很多插件（更多imagemin插件），这里只是简单介绍一下，要想压缩不同格式的图片，必须对应安装不同的插件，这里只安装了pngquant插件。按照nodejs的模块化思想，每个require只包含一个功能，这样就可以达到按需加载的目的。

九、gulp-livereload（自动刷新）

gulp-livereload安装：

npm install --save-dev gulp-livereload

当代码变化时，它可以帮助我们自动刷新页面，该插件最好配合谷歌浏览器，且要安装livereload chrome extension扩展插件，否则无效。

gulp-markdown安装：

npm install --save-dev gulp-markdown

当要把一个文件存储为不同版本时可以使用。比如在需要一个style.css同时你有需要一个style.min.css

gulp-rename安装

npm install --save-dev gulp-rename
